Put photos of the two of you here (jpg / png / webp).

Then list them in  /config.js  under  photos: [ ... ]
   photos: [
     "assets/photos/1.jpg",
     "assets/photos/2.jpg",
     "assets/photos/3.jpg"
   ]

If you leave the list empty, elegant gradient cards are shown instead,
so the site always looks finished.
